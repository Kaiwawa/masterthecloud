Compute Services

Virtual machines, serverless, and container orchestration on Azure.

Compute Service Decision Tree

Need full OS control?
Use Azure VMs (IaaS).
Just need to deploy code?
Use App Service (PaaS) or Functions (Serverless).
Need container orchestration?
Use AKS for complex apps, ACI for simple containers.

Storage Services

Object, block, and file storage solutions on Azure.

Blob Storage Access Tier Waterfall

Hot Tier Active Data / Lowest read cost / Highest storage cost
Cool Tier Infrequent Access (≥ 30 Days min)
Cold Tier Rare Access (≥ 90 Days min)
Archive Tier Long-term Retention (≥ 180 Days) / Lowest storage cost / Highest read cost

Networking

Virtual networks, hybrid connectivity, and traffic management.

NSG vs Azure Firewall Cheat Sheet

NSG (Network Security Group)
✦ Layer 3/4 filtering (IP, Port, Protocol)
✦ Applied to NIC or Subnet level
✦ Free to use
✦ Stateful (return traffic auto-allowed)
Azure Firewall
✦ Layer 3/4/7 filtering (FQDN, URL, TLS)
✦ Centralized, hub-level firewall
✦ Paid managed service
✦ Threat intelligence integration
